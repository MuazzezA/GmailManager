# GMAIL MANAGER

Gmail Manager is a user-friendly mobile application that enables users to manage their e-mails. All data is stored in the app's private storage, which is inaccessible to others. Without the user's permission, this data cannot be decrypted, and no information about the emails can be retrieved.

## Features

- **Manage emails** from a simple-to-use interface.
- **Secure data storage** in the app’s private storage, ensuring no access without user consent.
- Built with **React Native** for cross-platform compatibility.
- **User-friendly design** to ensure ease of use.

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/MuazzezA/GmailManager.git
    ```

2. Install the required dependencies:

    ```bash
    cd GmailManager
    yarn install
    ```

3. Run the app:

    - For iOS:

        ```bash
        npx react-native run-ios
        ```

    - For Android:

        ```bash
        npx react-native run-android
        ```
