# GMAIL MANAGER
> Gmail Manager is a simple to use mobile application that allows users to manage their e-mails. All data is stored in the app's private inaccessible storage. Without the consent of the person, this data cannot be decrypted and no information about the mails can be obtained. 


Gmail Manager, kullanıcıların e-postalarını yönetmelerini sağlayan kullanımı kolay bir mobil uygulamadır. Tüm veriler uygulamanın özel erişilemez depolama alanında saklanır. Kişinin izni olmadan bu verilerin şifresi çözülemez ve mailler hakkında hiçbir bilgi elde edilemez.
